`@dkharms`'s dotfiles.

## Prerequisites

Make sure that on your local machine installed following tools:
- `brew`;

## Some Info

Primary Theme – `tokyo-night`;

Package Manager – `brew`;

Terminal – `alacritty`;

Code Editor – `helix`;

## Get Started

Just run this command in your terminal:
```bash
sh init.sh
```

## Commands

Format python files:

```make
make format
```
