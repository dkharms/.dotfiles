### What's in the box?

* Primary Theme – `ayu-dark`;
* Package Manager – `brew`;
* Terminal – `alacritty`;
* Code Editor – `helix`;

### Prerequisites

Make sure that on your local machine installed following tools:
* `yadm` – https://yadm.io/;

### Get Started

Just run this command in your terminal:
```bash
yadm bootstrap
```
